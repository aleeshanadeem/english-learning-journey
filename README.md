## About

For the past **125 consecutive days**, I have practiced English every single day using **Duolingo**.

It is about building one of the most valuable skills for any Data Scientist, AI Engineer, or lifelong learner:

- Communication Skills
- Vocabulary
- Reading Speed
- Daily Discipline
- Learning Habit
- Confidence
